1. Basic requirments for the app.
    Browser
    xampp server
    mysql


2. Getting started with created the basic hirarchy of folders.
    Impliment the functionality by using php oops. 
    Connect the app with the mysql database for the data sync up.
    Create the basic template with the help of html/css , php and bootstrap.
    Take the referance from W3School for Templates.


3. Steps involved in running the app in localhost.
    Download and setup the xampp server on your mechine.
    Install the mysql database.
    Crea the the database with the name quiz_app.
    Connect it with the app.
    start the xampp server and start the mysql & apache.
    Then type http://localhost/quiz_php_mysql_html_css/ in the chrome browser.
